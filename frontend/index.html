Getting Started

Quick Start Guide:

  1. Choose Your Assistant - Click on any assistant card above to launch
  2. Start Asking - Enter your question or problem
  3. Get Results - Receive AI-powered answers with explanations
  4. Explore Further - Use advanced features like visualizations and synthesis

Backend Setup:

# Start the backend server
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# API will be available at:
# http://localhost:8000
# Docs: http://localhost:8000/docs

System Requirements:

  • Python 3.10+
  • 8GB RAM minimum (16GB recommended)
  • GPU recommended for SARAH (optional for EMMA/MAX)
  • Modern web browser (Chrome, Firefox, Safari, Edge)
