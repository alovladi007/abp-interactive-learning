Output:
(x³/3) + (3x²/2) + 2x + C

Output:
cos(2x)

Step-by-Step Solution:
1. Apply product rule: d/dx[sin(x)·cos(x)]
2. = cos(x)·cos(x) + sin(x)·(-sin(x))
3. = cos²(x) - sin²(x)
4. = cos(2x) (using double angle identity)

Output:
x = 2, x = 3
