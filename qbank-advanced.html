🔌 API Endpoints

POST /v1/quizzes

Create adaptive quiz session

GET /v1/quizzes/{quiz_id}/next

Get next adaptive question

POST /v1/quizzes/{quiz_id}/answers

Submit answer and update ability

POST /v1/admin/exposure/calibrate_sh/start

Start Sympson-Hetter calibration

Example Request

curl -X POST http://localhost:8000/v1/quizzes \ -H "Content-Type: application/json" \ -d '{ "tenant_id": "00000000-0000-0000-0000-000000000001", "filters": { "exam_code": "SAT-MATH", "num_questions": 30, "mode": "exam", "adaptive": true } }'

🚀 Deployment Instructions

Local Development

# Clone repository git clone https://github.com/alovladi007/abp-interactive-learning.git cd abp-interactive-learning/qbank-v10 # Start services docker-compose up -d # Initialize database docker-compose exec postgres psql -U qbank -d qbank \ -f /docker-entrypoint-initdb.d/01_core_schema.sql # Access services # API: http://localhost:8000 # Docs: http://localhost:8000/docs

Production Deployment

# Using Kubernetes kubectl apply -f k8s/ # Using Docker Swarm docker stack deploy -c docker-compose.yml qbank # Environment variables export DATABASE_URL=postgresql://... export REDIS_URL=redis://... export SECRET_KEY=your-secret-key
